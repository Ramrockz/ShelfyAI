# ShelfyAI

AI-powered inventory management system for makers and small businesses.

## Features

- 🧪 Ingredient tracking and management
- 🍳 Recipe composition and cost analysis
- 💰 Sales and expense analytics
- 🤖 AI-powered vision for receipt scanning
- 📊 Profit margin intelligence
- 🛒 Smart shopping list generation

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Auth & Database: Supabase
- Hosting: Vercel

## Quick Start

1. Clone the repository
2. Add your Supabase credentials to `auth.js`
3. Deploy to Vercel: `npx vercel`
